# Структура стилей CallTrack

## Обзор
Файлы стилей разделены на логические модули для лучшей поддержки и масштабируемости.

## Файлы

### `common.css`
Содержит общие стили, используемые на всех страницах:
- CSS переменные (цвета, размеры, тени)
- Базовые стили для body, header, navigation
- Общие компоненты (кнопки, формы, контейнеры)
- Медиа-запросы для мобильных устройств

### `tariffs.css`
Специфичные стили для страницы тарифов:
- Стили для поиска и фильтрации
- Выпадающие меню
- Карточки тарифов
- Сетка тарифов
- Мобильная адаптация

## Использование
```html
<!-- Подключение общих стилей -->
<link rel="stylesheet" href="styles/common.css">

<!-- Подключение специфичных стилей -->
<link rel="stylesheet" href="styles/tariffs.css">
```

## Принципы
- Использование CSS переменных для консистентности
- Модульная структура для легкого сопровождения
- Мобильная адаптация
- Семантические имена классов
