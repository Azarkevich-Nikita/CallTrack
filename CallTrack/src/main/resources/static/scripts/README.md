# JavaScript модули CallTrack

## Обзор
JavaScript код разделен на модули для лучшей организации и поддержки.

## Файлы

### `tariffManager.js`
Основной класс для управления тарифами:

#### Основные методы:
- `constructor()` - инициализация менеджера
- `initializeControls()` - настройка элементов управления
- `loadTariffs()` - загрузка данных с сервера
- `applyFilters()` - применение фильтров и сортировки
- `renderTariffs()` - отображение тарифов
- `createTariffCard()` - создание карточки тарифа

#### Фильтрация:
- `matchesSearch()` - поиск по названию
- `matchesType()` - фильтр по типу
- `matchesStatus()` - фильтр по статусу
- `matchesPriceRange()` - фильтр по цене

#### Сортировка:
- `sortTariffs()` - сортировка по различным критериям

#### Утилиты:
- `getTypeLabel()` - получение читаемого названия типа
- `isTariffActive()` - проверка активности тарифа
- `formatDate()` - форматирование даты
- `showError()` - отображение ошибок
- `showEmptyState()` - пустое состояние

## Использование
```html
<script src="scripts/tariffManager.js"></script>
```

- Объектно-ориентированный подход
